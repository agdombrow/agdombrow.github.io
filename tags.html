Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via the table tag, table rows with the tr tag, table cells with the td tag, and table headers with the th tag.

The tables below don't include all HTML tags, but they do include a few that you can use in practice.

Block Elements

Tag Name Purpose
<h1>–<h6> headings include a heading (levels 1–6)
<p> paragraph Defines a paragraph
<img> image Defines an image
<ol> list Defines an ordered list
<ul> list Defines an unordered list
<li> list item Defines a list item
<table> table Defines a table
<tr> table row Defines a table row
<th> table header Defines a table header
<td> table data Defines a table data

Inline Elements

Tag Name Purpose Example Result
<em> emphasized text make text italicized <p>My favorite food is <em>apples</em>.</p> apples
<strong> bold text make text bold <strong>I am strong!</strong> I am strong!
<a> link make hyperlink <a href="https://www.google.com">Google</a> Google
<img> image insert an image <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
<code> computer code Define a piece of computer code <code>SELECT * FROM users;</code> SELECT * FROM users;
